Privacy Policy

Last Updated: November 3, 2024

Your Privacy Matters
We are committed to protecting your privacy. This policy explains what data we collect, how we use it, and your rights regarding your data.

1. Information We Collect

ZH Anti Link Bot collects and stores the following information necessary for its operation:

Server Data

  • Server ID and name
  • Configuration settings (whitelisted domains, channels, roles)
  • Bot personality preferences and toggle settings
  • Maximum warning thresholds

User Data

  • User IDs of members who violate rules
  • Warning counts and violation history
  • Timestamps of violations
  • Detected link content (for logging purposes)

Message Data

  • Message content is scanned in real-time but not permanently stored
  • Only violation logs are retained (message ID, timestamp, user ID)
  • Messages are never shared or sold to third parties

2. How We Use Your Information

The collected data is used exclusively for:

  • Operating the bot's link detection and moderation features
  • Tracking user warnings and enforcing progressive punishments
  • Providing statistics and logs to server administrators
  • Maintaining bot configuration across sessions
  • Improving detection algorithms and performance

3. Data Storage and Security

We take data security seriously:

  • All data is stored locally per server
  • Data is encrypted and protected from unauthorized access
  • We implement industry-standard security measures
  • Regular backups are performed to prevent data loss
  • No data is shared with third-party services

4. Data Retention

Data retention policies:

  • Server configuration data is retained while the bot is in your server
  • Violation logs are retained for up to 30 days by default
  • User warning data persists until manually reset
  • All data is automatically deleted when the bot is removed from a server

5. Your Rights

As a server administrator, you have the right to:

  • Access all data stored for your server
  • Export your server's data using the export command
  • Request deletion of specific data
  • Reset user warnings and violation history
  • Remove the bot and have all data deleted

6. Data Sharing and Third Parties

We do NOT:

  • Sell your data to third parties
  • Share data with advertisers or marketers
  • Use your data for purposes other than bot operation
  • Track user activity across servers
  • Implement analytics or telemetry beyond basic bot statistics

7. Compliance with Discord

ZH Anti Link Bot operates in full compliance with:

  • Discord Terms of Service
  • Discord Developer Terms
  • Discord Privacy Policy
  • Discord Community Guidelines

8. Children's Privacy

The bot does not knowingly collect information from users under 13. If you believe a child has provided data to the bot, contact us immediately.

9. International Users

By using the bot, you consent to data processing in accordance with this privacy policy, regardless of your location.

10. Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected with an updated "Last Updated" date. Continued use of the bot constitutes acceptance of changes.

11. Data Breach Notification

In the unlikely event of a data breach, we will:

  • Notify affected servers immediately
  • Provide details about the nature of the breach
  • Take immediate action to secure systems
  • Cooperate with any necessary investigations

12. Contact and Data Requests

For privacy concerns, data access requests, or deletion requests:

  • Join our support Discord server
  • Contact us through official channels
  • Allow up to 7 days for data request processing

13. Transparency

We believe in full transparency. If you have questions about how your data is handled, we're happy to provide detailed explanations.

Questions?
If you have any questions about this privacy policy or how we handle your data, please reach out through our support server.
